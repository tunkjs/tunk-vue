## tunk-vue


**tunk-vue** 顾名思义，是让tunk与vue可以一起工作的组件，负责定义视图层如何 **触发Action** 及如何 **将新状态注入到视图组件**。


### 安装
````javascript
npm install tunk-vue -S
````
